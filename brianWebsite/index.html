Latitude vs. Climate Analysis

Background

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. Weather factors examined included: temperature, humididty, cloudiness, and wind speed. Further exploration of these factors can be found in the 'Plots' section above or 'Visualizations' section to the right.
